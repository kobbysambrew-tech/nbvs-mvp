NBVS Ghana — Verification API

Fast • Secure • Automated Background Checks for Developers & Agencies

Overview

The NBVS API allows approved businesses, employers, and government agencies to verify Ghanaian identity, criminal history, credit score, and driving status programmatically using a secure API key.

Base URL: https://nbvsghana.com/api/verify

Authentication

All API requests require an NBVS API Key. Keys are created in the NBVS Admin Dashboard by Superadmin only.

Header format:

{
  "Authorization": "Bearer YOUR_API_KEY"
}

Verify Person

POST /api/verify

Verify using either:

  • Full name
  • NIA ID / Ghana Card number

Example Request

POST https://nbvsghana.com/api/verify
Headers:
  Authorization: Bearer nbvs_6aJx91K...
Body:
{
  "query": "Ama Serwaa"
}

Example Response

{
  "success": true,
  "result": {
    "name": "Ama Serwaa",
    "nia": "GHA-2039485",
    "dob": "1994-07-12",
    "region": "Greater Accra",
    "criminal": "Clean",
    "driving": "Valid",
    "credit": 750,
    "status": "Verified"
  }
}

Error Responses

Invalid key:

{
  "success": false,
  "error": "Invalid API key"
}

No record found:

{
  "success": false,
  "error": "No match found"
}
