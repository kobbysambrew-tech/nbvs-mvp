NBVS API Documentation

National Background Verification System • Secure • Trusted • Fast

Overview

The NBVS API allows approved companies, employers, and government agencies to perform instant background verification using a secure API key.

All requests are authenticated using your API Key generated in the admin dashboard.

Get API Key →

Base Endpoint

GET https://your-vercel-domain.vercel.app/api/verify?key=API_KEY&nia=NIA_NUMBER

Parameters

key – Your assigned API key (required)

nia – National ID number to verify (required)

debug – Return detailed logs (optional)

Example Request (JavaScript)

const url = "https://your-domain/api/verify?key=YOUR_KEY&nia=GHA-1234567"; fetch(url) .then(r => r.json()) .then(data => console.log(data));

Example Response

{ "name": "Kofi Mensah", "nia": "GHA-1234567", "dob": "1995-02-17", "region": "Greater Accra", "criminal": "None", "driving": "Valid", "credit": "Good", "status": "Verified" }

Error Responses

{ "error": "Invalid API key" } { "error": "Verification not found" } { "error": "Rate limit exceeded (per minute)" }

Rate Limits

Each API key has system-level limits to prevent abuse:

  • 10 requests / minute
  • 200 requests / day

Custom limits can be configured per client.

Security

All requests are logged in audit_logs and search_logs. Only the Superadmin can view client activity.

Private data is encrypted and served through HTTPS only.
