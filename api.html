NBVS Ghana — Developer API

Secure · Encrypted · Verified Identity Access

Overview

The NBVS API allows approved partners (employers, agencies, platforms) to verify Ghanaian identity data such as:

  • Full Name
  • NIA Number
  • Date of Birth
  • Driving Status
  • Criminal Status
  • Region / Address
  • Credit Score
  • Verification Status

Access requires an API key generated inside your NBVS Admin panel.

Open Admin Panel

Authentication

All requests must include this header:

Authorization: Bearer YOUR_API_KEY

Never share your API key publicly.

Endpoint: Search Record

GET https://nbvsghana.com/api/search?query=ama%20serwaa

Example Request:
curl -X GET "https://nbvsghana.com/api/search?query=ama serwaa" \
  -H "Authorization: Bearer nbvs_6a8f9example"

Example Response:
{
  "success": true,
  "results": [
    {
      "name": "Ama Serwaa",
      "nia": "GHA-2039485",
      "dob": "1994-07-12",
      "region": "Greater Accra",
      "address": "Accra, Ghana",
      "criminal": "Clean",
      "driving": "Valid",
      "credit": 750,
      "status": "Verified"
    }
  ]
}

Error Responses

401 — Missing or invalid API key
403 — Permission denied (wrong role)
429 — Rate limit exceeded
500 — Server error

Rate Limits

Each API key has a rate limit of:

  • 60 requests per minute
  • 2,000 requests per day

Higher limits available upon request.

Regions Supported

NBVS supports all regions of Ghana:

  • Greater Accra
  • Ashanti
  • E. Region
  • Northern
  • Upper East / West
  • Central
  • Volta
  • Western North + more
